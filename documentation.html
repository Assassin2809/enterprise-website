Identifying and Defining

Audience: New students or visitors seeking to navigate the school easily. This website is the go-to-tool for any student that like interactive websites that help find them the information they need. For example, if you want to find where they library is on the school map, this is the website for you!, if you want to get a insider view of the school through our pristine 4k HD 360 Cameras, then this is the website for you!

Information: A digital map displaying room locations and key areas of the school. It includes a brand-new up to date 360 view of the school with 4k 360 cameras. It also includes a comprehensive guide that explains how to use the website.

Safety and Ethics: Ensuring user data privacy and accessibility compliance by not taking in any information from the user and incorporating visual views of the school.The website is safe for everyone to use and it follows all the rules and is fair to everyone.

Engagement: Interactive features, visual cues,sleek animations and intuitive design which engages users. The interactive features includes a wide array of buttons, and search functionality to engage the user. Visual cues include the pointing of an arrow on the school map that helps the user figure out where the room they want is.
